Projects

Jbops

Live Link GitHub

JBOPS is a clone of Spotify's web player, built using React/Redux for the frontend, Ruby on Rails for the backend API, and AWS S3 cloud storage.

My favorite part was learning to build a media player using HTML5's audio element and watching my application come to life. This was my first fullstack project and the time that I spent building it was invaluable. Eventually, I would like to go back and refactor my code to make it more efficient and DRY.

Connect 4riends

Live Link GitHub

During the social distancing stages of COVID-19, I teamed up with 3 other talented software engineers to build a platform to play Connect 4 against friends locally, online, or against a computer. Connect 4riends was developed using React/Redux, MongoDB, Express.js, Node.js, and Socket.io.

As the team lead and flex developer, I was responsible for strategic planning and task delegation, leading daily standup meetings, assisting my backend and frontend developers, and building the logic and functionality for both local and online modes.

Bill Mates

Live Link GitHub

Bill Mates is a web application that allows users to assign items from their bill or receipt to individual guests in order to calculate the total amount owed by each person. In order to test my proficiency with vanilla JavaScript and HTML/CSS, no additional libraries or frameworks were used.

Give it a try after covering the bill with friends on a night out!

About
